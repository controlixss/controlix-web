# 📊 Controlix - Dato Mata Relato

**Controlix** es una solución de gestión financiera diseñada específicamente para dueños de bares y restaurantes que buscan optimizar sus márgenes de beneficio.

## 🚀 El Proyecto
Esta es la landing page oficial donde presentamos nuestra propuesta de valor: transformar los tickets y facturas en decisiones estratégicas mediante auditorías personalizadas.

### Servicios:
* **Pack Gestión Base (97€):** El motor de Controlix listo para usar.
* **Pack Auditoría Total (157€):** Implementación "llave en mano" con carga de datos y diagnóstico estratégico.

## 🛠️ Tecnologías
* HTML5 / CSS3 (Optimizado para dispositivos móviles).
* Integración con WhatsApp Business para atención al cliente.
* Stripe (En proceso de integración).

## 📈 Visión
En la hostelería, muchos negocios mueren facturando mucho pero ganando poco. Controlix nace para eliminar la ceguera financiera.
